UC Davis Sample Pamphlet

Finding balance between style and content

For one of my classes, I had to create a sample pamphlet for the Design Department at UC Davis, using a select number of images and text to successfully advertise the program. With this project, my main goal was to implement a clean style through a grid system that would be easy for visitors to read and would focus more on the actual results of the program than the pamphlet itself.
